# 3D Web 魔法游乐场

你好！👋 这是一个探索现代 Web 3D 技术的游乐场项目。它结合了经典的 Three.js 场景和一个基于 Babylon.js 的实验性手势控制体验。

我创建这个项目是为了深入研究 WebGL、粒子系统以及计算机视觉交互技术。

## 项目内容

### 🎄 手势控制圣诞树 (Babylon.js + MediaPipe)

这是最酷的部分。它利用你的网络摄像头追踪手部动作，让你能实时通过手势控制 3D 粒子圣诞树。完全不需要键盘鼠标！

**体验方式：** 打开 `babylon.html`（或点击应用首页的链接）。

**魔法功能：**
*   **移动手掌**：摄像头会跟随你的手。向左或向右移动手掌，可以旋转观察圣诞树。
*   **张开/握拳**：这是我最喜欢的功能 —— **张开手掌**，树上的粒子会像呼吸一样向外扩散；**握紧拳头**，粒子会迅速收紧回原形。
*   **捏合放烟花**：这个超解压！捏合拇指和食指，光标处就会炸开绚丽的烟花。💥
*   **视觉反馈**：屏幕上有一个青色的 3D 光标会实时跟随你的手，让你知道系统“看”到的手在哪里。

### 📦 经典丰富场景 (Three.js)

这是最初的学习项目。它是一个展示 3D 渲染基础的综合场景。

*   **几何体与材质**：立方体、球体、圆环... 包含金属、玻璃、自发光等多种材质。
*   **动态光源**：彩色的点光源在场景中环绕运动。
*   **物理动画**：弹跳的球体和浮动的几何体。
*   **阴影**：完整的柔和阴影映射。

## 快速开始

这是一个标准的 Vite 项目，启动非常简单：

```bash
# 安装依赖 (我们使用 pnpm, 但 npm/yarn 也可以)
pnpm install

# 启动！
pnpm dev
```

然后访问 `http://localhost:5173` 即可。

## 技术栈

*   **渲染引擎**: Babylon.js (圣诞树) & Three.js (经典场景)。
*   **视觉/AI**: MediaPipe Hands (提供丝滑的手部追踪)。
*   **构建工具**: Vite。

## 注意事项

*   **摄像头权限**: Babylon 演示需要摄像头权限。别担心，所有计算都在你的浏览器**本地运行**，没有任何视频画面会被上传到服务器。
*   **性能**: 粒子树包含 4000+ 个粒子。如果你的风扇开始转了，就当是圣诞气氛到了吧。🎅

玩得开心！
